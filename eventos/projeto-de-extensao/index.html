O que é o Projeto de Extensão?

O Projeto de Extensão é uma disciplina obrigatória para os alunos do 1º e 2º semestres na FAM. Ela conecta o aprendizado teórico com a prática, permitindo que você use o conhecimento adquirido em sala de aula para impactar positivamente a comunidade. É uma oportunidade de contribuir para causas sociais e filantrópicas, enquanto desenvolve suas habilidades acadêmicas e pessoais.

Todas as informações importantes sobre o Projeto de Extensão, incluindo regulamentos e prazos, estão disponíveis no plano de aula da disciplina, acessível no PORTAL DO ALUNO.
